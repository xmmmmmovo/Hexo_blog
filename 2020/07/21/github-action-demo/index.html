GithubAction

官网的第一段，便讲明了此功能的用途和作用。

在 GitHub 操作 的仓库中自动化、自定义和执行软件开发工作流程。 您可以发现、创建和共享操作以执行您喜欢的任何作业（包括 CI/CD），并将操作合并到完全自定义的工作流程中。

白嫖Github服务器用来部署自己的web应用。部署后链接格式：<用户名>.github.io/<仓库名>
